  <action>
    1. Create `Dockerfile` at project root (multi-stage):
       ```
       FROM node:22-alpine AS builder
       WORKDIR /app
       COPY package.json package-lock.json ./
       RUN npm ci
       COPY . .
       RUN npm run build
       RUN npm prune --production

       FROM node:22-alpine
       WORKDIR /app
       COPY --from=builder /app/build build/
       COPY --from=builder /app/node_modules node_modules/
       COPY --from=builder /app/package.json .
       EXPOSE 3000
       ENV NODE_ENV=production
       CMD ["node", "build"]
       ```

    2. Create `docker-compose.yml` at project root:
       - 3 services: app (build: ., ports 3000:3000, depends_on postgres+redis with service_healthy, env_file .env, environment DATABASE_URL and REDIS_URL), postgres (postgres:16-alpine, POSTGRES_DB=llmtokenhub, POSTGRES_USER=postgres, POSTGRES_PASSWORD=postgres, volume postgres_data, healthcheck pg_isready), redis (redis:7-alpine, volume redis_data, healthcheck redis-cli ping)
       - Named volumes: postgres_data, redis_data
       - Exact config from 05-RESEARCH.md Docker Compose section

    3. Create `.dockerignore`:
       ```
       node_modules
       .svelte-kit
       build
       .env
       .git
       .planning
       docs
       *.md
       !package.json
       ```

    4. Update `.env.example` — ensure it includes ALL env vars:
       - DATABASE_URL=postgresql://postgres:postgres@localhost:5432/llmtokenhub
       - REDIS_URL=redis://localhost:6379
       - ENCRYPTION_KEY= (32-byte hex string for AES-256-GCM)
       - SMTP_HOST=, SMTP_PORT=587, SMTP_USER=, SMTP_PASS=, SMTP_FROM=
       - GOOGLE_CLIENT_ID=, GOOGLE_CLIENT_SECRET=
       - GITHUB_CLIENT_ID=, GITHUB_CLIENT_SECRET=
       - BASE_URL=http://localhost:3000
       - Comments explaining each variable
