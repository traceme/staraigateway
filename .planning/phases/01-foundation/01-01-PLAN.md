       - `app_email_verifications`: id (uuid, pk), user_id (uuid, FK), token (text, unique, not null), expires_at (timestamp), created_at
       - `app_password_resets`: id (uuid, pk), user_id (uuid, FK), token (text, unique, not null), expires_at (timestamp), used (boolean, default false), created_at

       - `app_org_members`: id (uuid, pk), org_id (uuid, FK), user_id (uuid, FK), role (text, not null -- 'owner'|'admin'|'member'), created_at
       - Add unique constraint on (org_id, user_id) for app_org_members
