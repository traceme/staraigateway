<task type="auto">
  <name>Task 1: API key generation, hashing, and CRUD server logic</name>
  <files>src/lib/server/api-keys.ts</files>
  <action>
Create src/lib/server/api-keys.ts with the following functions:

1. `generateApiKey(): { fullKey: string, prefix: string, hash: string }`
   - Generate 48 random bytes using Node.js crypto.randomBytes
   - Encode as base62 or hex to create the key body
   - Full key format: `sk-th-${body}` (body should be ~48 chars)
   - Prefix: first 12 characters of full key (e.g., 'sk-th-abc123') for display
   - Hash: SHA-256 of the full key using Node.js crypto.createHash (NOT oslo -- keep it simple, this is a one-way hash not session management)
   - Return all three values

2. `createApiKey(orgId: string, userId: string, name: string): Promise<{ key: ApiKeyMetadata, fullKey: string }>`
   - Call generateApiKey() to get key material
   - Insert into appApiKeys: id (crypto.randomUUID()), orgId, userId, name, keyPrefix, keyHash, isActive: true
   - Return the inserted row metadata (without hash) AND the fullKey (this is the only time full key is available)

3. `getUserApiKeys(orgId: string, userId: string): Promise<ApiKeyMetadata[]>`
   - Select all keys for this user in this org, ordered by createdAt desc
   - Return: id, name, keyPrefix, isActive, lastUsedAt, createdAt (never return keyHash)

4. `revokeApiKey(id: string, orgId: string, userId: string): Promise<boolean>`
   - Set isActive = false where id matches AND orgId matches AND userId matches
   - Return true if a row was updated, false otherwise
   - Do NOT delete the row (keep for audit trail)

5. `validateApiKeyFromHash(keyHash: string): Promise<{ apiKey: ApiKeyRow, user: UserRow, org: OrgRow } | null>`
   - Look up key by keyHash, join with appUsers and appOrganizations
   - Return null if key not found, not active, or org not found
   - Update lastUsedAt timestamp on successful validation
   - This function is used by the gateway (Plan 03) to authenticate incoming requests

Define a `ApiKeyMetadata` type locally: Pick from ApiKey excluding keyHash.

Use Drizzle ORM for all queries. Follow the existing patterns from the codebase (db.select().from()...).
