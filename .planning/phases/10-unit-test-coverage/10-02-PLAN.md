6. **Cache miss, DB miss (inactive key)** — DB returns empty array → returns null
7. **Rate limit cascade: per-key override** — DB row has `rpmLimit: 100`, `defaultRpmLimit: 50` → `effectiveRpmLimit` is 100
8. **Rate limit cascade: org default** — DB row has `rpmLimit: null`, `defaultRpmLimit: 50` → `effectiveRpmLimit` is 50
9. **Rate limit cascade: both null** — DB row has `rpmLimit: null`, `defaultRpmLimit: null` → `effectiveRpmLimit` is null
10. **Redis failure gracefully degrades** — `getRedis()` returns `{ get: vi.fn().mockRejectedValue(new Error('conn')) }`, verify falls through to DB lookup
